# Notification Sounds

This directory contains audio files for notification sounds.

## Required Sound Files

Place the following MP3 files in this directory:

1. **notification.mp3** - General notification sound (plays when new messages arrive)
2. **mention.mp3** - Mention notification sound (plays when you're mentioned)
3. **sent.mp3** - Message sent confirmation sound

## Recommended Specifications

- **Format**: MP3
- **Duration**: 0.5 - 2 seconds
- **Volume**: Pre-normalized to comfortable levels
- **Sample Rate**: 44.1kHz or 48kHz
- **Bit Rate**: 128-192 kbps

## Free Sound Resources

You can find free notification sounds at:
- [Freesound.org](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)
- [Notification Sounds](https://notificationsounds.com/)

## Example Sounds

For testing purposes, you can use:
- notification.mp3: Short "ding" or "pop" sound
- mention.mp3: Slightly longer "chime" sound
- sent.mp3: Subtle "whoosh" or "click" sound

## Usage

Sounds are automatically loaded by the application from:
- `/sounds/notification.mp3`
- `/sounds/mention.mp3`
- `/sounds/sent.mp3`

No code changes needed once files are placed here.
